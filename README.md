# Инструкция по запуску

1 - Скачать LTS версию Node.js

2 - Клонировать данный репозиторий на свой ПК

3 - Открыть в IDE

4 - Прописать
```sh
npm install
```
5 - Прописать
```sh
npm run dev
```
6 - Готово
